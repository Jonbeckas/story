An atom scriptwriting plugin, made for LUMOS.

# Syntax

| Syntax        | Explenation           |
| ------------- |:-------------:|
| /\*...\*/     | Title          |
| // ENV TIME 'Name'      | Scene      |
| EXT\|IN | Enviroment      |
|DAY\|NIGHT\|EVENING\|MORNING\|NOON|Time|
|@Name|Character|
|""|Speech|
|"(directing Instruction)"|Directing Instruction|
|#comment|Comment|
|+Object|Off camera objects|
|*object|set objects|
